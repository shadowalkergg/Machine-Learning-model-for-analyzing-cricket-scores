# Machine-Learning-model-for-analyzing-cricket-scores
Predict Men's T20 cricket scores using SVR with Gaussian Kernel (RBF). Trained on 3,000+ matches, using 15-over score, wickets lost, and venue.  Average error: 11 runs Python 3.x, scikit-learn, Matplotlib, pandas, numpy. Submit a pull request to improve the model or add features.
